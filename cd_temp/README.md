# source

```
./cd_temp.sh
source ./cd_temp.sh
```

所谓 source，就是让 script 在当前 shell 内执行、 而不是产生一个 sub-shell 来
执行。由于所有执行结果均在当前 shell 内执行、而不是产生一个 sub-shell 来执行。
